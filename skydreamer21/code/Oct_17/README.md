# 최대힙과 최소힙
## 최대 힙
부모 노드의 값이 자식 노드의 값보다 크거나 같은 완전 이진 트리

## 최소 힙
부모 노드의 값이 자식 노드의 값보다 작거나 같은 완전 이진 트리

# 요소 추가와 삭제
## 추가
마지막에 추가를 하고 부모와 비교하며 자기 자리를 찾을 때까지 올라가는 것을 반복한다.

## 삭제
첫번째와 마지막을 바꾸고 마지막을 자식과 비교하여 자기 자리를 찾을 때까지 내려가는 것을 반복
